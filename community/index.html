Community

JIMDB Community

JIMDB is development in the open. We welcome more contributors join us. Here are some of channels we use to communicate and contribute

Mailing list: jimdb@groups.io – discussion around JIMDB usage, community support and developer discussion.

Slack: jimdb.slack.com

Twitter: @jimdb

Issue tracker: Use the Github issue tracker to file bugs and features request. If you need support, please send your questions to the jimdb@groups.io mailing list rather than filing a GitHub issue.

Contributing
We welcome community contributions! If you are planning on making more elaborate or controversial changes, please discuss them on the mailing list or on GitHub before sending a pull request.

Project Governance
JIMDB is an independent open-source project and not controlled by any single company. More details please refer to Governance.md.

Code of Conduct
To make JIMDB a welcoming and harassment-free experience for everyone, we follow the CNCF Code of Conduct.
